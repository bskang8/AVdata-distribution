# AdvSim — 자율주행을 위한 안전 위험 시나리오 자동 생성

## 출처
- **저자**: J. Wang, A. Pun, J. Tu, S. Manivasagam, A. Sadat, S. Casas, M. Ren, R. Urtasun
- **연도**: 2021
- **학술대회**: CVPR 2021
- **파일**: `literature/papers/R12_AdvSim_CVPR2021.pdf`

---

## 핵심 아이디어

기존 실제 로그의 **에이전트 궤적을 물리적으로 타당하게 변조**하여 안전 위험 시나리오를 대규모 자동 생성한다. 데이터 수집 없이 기존 데이터만으로 다양한 위험 시나리오를 합성한다.

### 작동 원리

```
실제 주행 로그 (ego + 주변 에이전트 궤적)
    │
    ▼  Adversarial 최적화
에이전트 궤적 변조 (물리 제약 만족 조건 하에)
    │
    ├─ 차선 변경 타이밍 조정
    ├─ 속도 프로파일 수정 (급가속/급감속)
    ├─ 보행자 돌출 시점 변경
    │
    ▼
Safety-Critical 시나리오 (충돌 직전, 급제동 상황 등)
```

### 물리 제약 조건

- 최대 가속도 / 감속도 제한
- 차선 경계 준수
- 에이전트 간 최소 간격 유지 (비현실적 겹침 방지)

### 결과

- 기존 로그 대비 **3.8배 더 많은 safety-critical 이벤트** 생성
- 자율주행 모델의 위험 시나리오 견고성 테스트에 활용

---

## 장단점

**장점**
- 실제 물리 로그 기반 → 현실적인 시나리오 생성
- 새 데이터 수집 불필요 → 비용 절감
- 물리 제약으로 비현실적 시나리오 배제
- 다양한 에이전트 유형(차량, 보행자, 자전거)에 적용 가능

**단점**
- 센서 데이터(LiDAR, 카메라)가 있는 원본 로그 필요
- 캡션 텍스트만 있는 경우 직접 적용 불가
- 환경 조건(날씨, 조명) 변조는 별도 시뮬레이터 필요

## 프로젝트 적용 포인트

### Gap-4 / Gap-6 → EXP-002 Phase D

Phase A/B에서 식별된 희귀 시나리오에 대해 실제 수집 대신 합성으로 보완:

| 시나리오 | 실제 수집 난이도 | AdvSim 적용 |
|---------|--------------|------------|
| "야간 역주행 차량 회피" | 매우 높음 | 기존 야간 로그에서 역주행 궤적 합성 |
| "급제동 후 연쇄 반응" | 높음 | 앞차 감속 타이밍 변조 |
| "보행자 돌발 횡단" | 높음 | 보행자 출현 시점 변조 |

현재 프로젝트에 raw 센서 로그가 있다면 EXP-002 Phase D에서 직접 활용 가능.
